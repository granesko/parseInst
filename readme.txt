Для работы скрипта inst_pool требуется python версии 3 и выше.
Содержимое архива нужно разархивировать в папку, откуда будет запускаться терминал.
Требуемые библиотеки:requests, multiprocessing, datetime, pandas    

Для запуска скрипта необходимо:
Файл links_instagram1.txt должен содержать список ссылок, которые пользователь хочет обработать.

Запуск:
В командной строке windows (win+R cmd) прописать путь до папки где находится скрипт, например:
cd "C:\PythonScripts"
затем ввести: 
python "название скрипта".py в нашем случае facebookscraper.py, затем выбрать метод:
-D -s [дата в формате %Год-%месяц-%число] метод обработки по дате, получит все посты до указанной даты.
-P -v [количество постов] метод обработки по количеству постов. Соберёт информацию по последним постам в указанном количестве.
Пример запуска: 
python facebookscraper -D -s 2020-02-01 (Все посты до 1-го февраля 2020 года)
python facebookscraper -P -v 50 (Последние 50 постов)

Результаты будут сохраняться в формате .json для каждой ссылки свой файл имя_артиста.json, в папку res_inst. 

ВАЖНО!
Не забывайте указывать параметры, по умолчанию стоят либо 300 постов, либо за последние 12 месяцев все посты, причем программа будет завершаться, когда одно из условий будет выполняться

Примерная статистика:
5 потоков с пазуами 20 минут за 2 часа 420 ссылок по 50 постов за ссылку
